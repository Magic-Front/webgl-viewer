frame

  • frame(target: "all" | Box3 | Sphere | Object, angle?: FrameAngle, duration?: number): void

  • Moves and rotates the camera so that target is well framed.

    Parameters

    • target: "all" | Box3 | Sphere | Object

      Vim or Three object to frame, all to frame the whole scene, undefined has no effect.

    • angle: FrameAngle = 'none'

      None will not force any angle, Center will force camera.y = object.y, providing an angle will move the camera so it is looking down at object by the provided angle.

    • duration: number = 0

      Duration of the lerp effect.

    Returns void

    Implementation of ICamera.frame
